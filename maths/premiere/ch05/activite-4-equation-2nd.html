Activité 4 – Résolution d'équation du 2nd degré SITUATION PRO

Chapitre 5 — Polynômes degré 2 | 1ère Bac Pro | Mathématiques | ⏱ 30 min

Dernière mise à jour : 5 mai 2026, format manuel scolaire

Objectifs :
  • Calculer le discriminant Δ d'une équation du 2nd degré
  • Trouver les racines
  • Étudier le signe d'un polynôme

Situation – dimensionner un panneau cintré

Une ébéniste fabrique un panneau cintré dont la fonction de hauteur est : f(x) = x² − 5x + 4 (en cm), x étant la position horizontale entre 0 et 5 cm.

Problématique : En quels points le panneau a-t-il une hauteur nulle, et quel est son point le plus bas ?

Question 1 REA

Calculer le discriminant Δ.

Δ = b² − 4ac = (−5)² − 4 × 1 × 4 = 25 − 16 = 9. Δ > 0 → 2 racines réelles.

Question 2 REA

Calculer les racines de f(x) = 0.

x = (5 ± 3)/2 → x_1 = 1, x_2 = 4.

Le panneau a une hauteur nulle en x = 1 cm et x = 4 cm.

Question 3 REA

Calculer le sommet de la parabole.

x_s = −b/(2a) = 5/2 = 2,5 cm.

f(2,5) = 6,25 − 12,5 + 4 = −2,25 cm.

Le point le plus bas est à 2,25 cm sous l'axe (panneau bombé vers le bas entre x = 1 et x = 4).

Question 4 ANA

Étudier le signe de f(x).

a = 1 > 0 → la parabole est tournée vers le haut.

  • x < 1 ou x > 4 : f(x) > 0.
  • 1 < x < 4 : f(x) < 0.
  • x = 1 ou x = 4 : f(x) = 0.

Question 5 REA

Factoriser f(x) en utilisant les racines.

f(x) = (x − 1)(x − 4).

Vérification : (x − 1)(x − 4) = x² − 4x − x + 4 = x² − 5x + 4 ✔

Question 6 ANA

Si on translate la fonction de 5 cm vers le haut : g(x) = f(x) + 5. Combien de racines ?

g(x) = x² − 5x + 9. Δ = 25 − 36 = −11 < 0 → aucune racine réelle.

La parabole ne touche plus l'axe x : elle est entièrement au-dessus.

Question 7 VAL

Pour quelle translation k la fonction f(x) + k aurait-elle une seule racine (parabole tangente à l'axe) ?

Δ = 0 ↔ b² − 4 a (c + k) = 0 → 25 − 4(4 + k) = 0 → k = 2,25.

(C'est exactement la valeur opposée du minimum f_min = −2,25.) f(x) + 2,25 a son sommet sur l'axe.

Question 8 COM

Résumer en 4 lignes les différents cas selon le signe de Δ.

Discriminant Δ et racines
• Δ > 0 : 2 racines réelles distinctes (parabole coupe l'axe en 2 points).
• Δ = 0 : 1 racine double (parabole tangente à l'axe).
• Δ < 0 : aucune racine réelle (parabole entière au-dessus ou en-dessous).
Pour ax² + bx + c = 0 avec Δ > 0 : x = (−b ± √Δ)/(2a).

Pour aller plus loin (bonus)

Pour quelles valeurs de m l'équation x² − 5x + m = 0 a-t-elle 2 solutions positives ?

2 solutions positives ⇔ Δ > 0 (2 racines), somme > 0 (5 ✓), produit > 0.

Δ = 25 − 4m > 0 → m < 25/4 = 6,25. Produit = m > 0.

Réponse : 0 < m < 6,25.

À retenir
  • Discriminant Δ = b² − 4ac. Selon le signe : 2 / 1 / 0 racines.
  • Racines : x = (−b ± √Δ) / (2a) si Δ > 0.
  • Factorisation : a(x − x_1)(x − x_2).
  • Signe : suit la règle « du signe de a hors des racines, du signe contraire entre ».
