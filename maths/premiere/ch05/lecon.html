Chapitre 5 – Fonctions polynômes de degré 2

1ère Bac Pro | Algèbre – Analyse | Mathématiques

Objectifs du chapitre

  • Reconnaître une fonction polynôme de degré 2 et identifier ses coefficients
  • Déterminer le sommet et l'axe de symétrie de la parabole
  • Calculer le discriminant et les racines éventuelles
  • Dresser le tableau de signes et de variations
  • Résoudre des problèmes concrets modélisés par une fonction du second degré

1. Introduction — Les paraboles dans la vie courante

Situation professionnelle — Charpentier

Contexte : Un charpentier conçoit une arche en bois pour l'entrée d'un jardin. La forme de l'arche suit une courbe parabolique. Pour garantir le passage d'un véhicule, il doit calculer la hauteur maximale de l'arche et vérifier que la largeur est suffisante à une certaine hauteur.

La hauteur de l'arche est modélisée par \(h(x) = -x^2 + 4x\), où \(x\) est la distance horizontale en mètres. Ce type de fonction s'appelle une fonction polynôme de degré 2.

2. Forme générale et reconnaissance

Définition — Fonction polynôme de degré 2 :
Une fonction polynôme de degré 2 est une fonction \(f\) définie sur \(\mathbb{R}\) par : \[f(x) = ax^2 + bx + c\] où \(a\), \(b\) et \(c\) sont des nombres réels avec \(a \neq 0\).
  • \(a\) est le coefficient dominant (il détermine l'ouverture de la parabole)
  • \(b\) est le coefficient du terme en \(x\)
  • \(c\) est le terme constant (c'est l'ordonnée à l'origine : \(f(0) = c\))
Exemple — Identifier les coefficients
Fonction\(a\)\(b\)\(c\)
\(f(x) = 2x^2 - 8x + 6\)2−86
\(g(x) = -x^2 + 4x\)−140
\(h(x) = 3x^2 + 5\)305
Propriété — Forme de la parabole :
La courbe représentative d'une fonction du second degré est une parabole.
  • Si \(a > 0\) : la parabole est ouverte vers le haut (forme en U) — elle admet un minimum.
  • Si \(a < 0\) : la parabole est ouverte vers le bas (forme en ∩) — elle admet un maximum.

3. Sommet et axe de symétrie

Définition — Sommet de la parabole :
Le sommet \(S\) de la parabole est le point le plus bas (si \(a > 0\)) ou le plus haut (si \(a < 0\)) de la courbe. C'est le point où la fonction atteint son extremum.
Coordonnées du sommet
\[x_S = \dfrac{-b}{2a} \qquad\text{et}\qquad y_S = f(x_S)\]
Propriété — Axe de symétrie :
La parabole admet un axe de symétrie : c'est la droite verticale d'équation \(x = x_S = \dfrac{-b}{2a}\).
Méthode — Calculer le sommet :
1
Identifier \(a\), \(b\), \(c\).
2
Calculer \(x_S = \dfrac{-b}{2a}\).
3
Calculer \(y_S = f(x_S)\) en remplaçant \(x\) par \(x_S\) dans l'expression de \(f\).
Exemple — Sommet de \(f(x) = 2x^2 - 8x + 6\)

\(a = 2\), \(b = -8\), \(c = 6\).

\(x_S = \dfrac{-(-8)}{2 \times 2} = \dfrac{8}{4} = 2\).

\(y_S = f(2) = 2(4) - 8(2) + 6 = 8 - 16 + 6 = -2\).

Le sommet est \(S(2\;;\;-2)\). Comme \(a = 2 > 0\), c'est un minimum.

4. Discriminant et racines

Définition — Discriminant :
Le discriminant de l'équation \(ax^2 + bx + c = 0\) est le nombre : \[\Delta = b^2 - 4ac\]
Propriété — Nombre de solutions selon \(\Delta\) :
  • Si \(\Delta > 0\) : l'équation admet deux solutions distinctes : \[x_1 = \dfrac{-b - \sqrt{\Delta}}{2a} \qquad\text{et}\qquad x_2 = \dfrac{-b + \sqrt{\Delta}}{2a}\]
  • Si \(\Delta = 0\) : l'équation admet une solution double : \[x_0 = \dfrac{-b}{2a}\]
  • Si \(\Delta < 0\) : l'équation n'admet aucune solution réelle.
Méthode — Résoudre \(ax^2 + bx + c = 0\) :
1
Identifier \(a\), \(b\), \(c\).
2
Calculer \(\Delta = b^2 - 4ac\).
3
Selon le signe de \(\Delta\) : appliquer la formule des racines ou conclure « pas de solution ».
Exemple — Résoudre \(2x^2 - 8x + 6 = 0\)

\(a = 2\), \(b = -8\), \(c = 6\).

\(\Delta = (-8)^2 - 4 \times 2 \times 6 = 64 - 48 = 16 > 0\) : deux solutions.

\(\sqrt{\Delta} = 4\).

\(x_1 = \dfrac{8 - 4}{4} = 1\) et \(x_2 = \dfrac{8 + 4}{4} = 3\).

Attention — Erreurs fréquentes :
  • Ne pas oublier le signe de \(b\) dans \(-b\) : si \(b = -8\), alors \(-b = 8\).
  • Le discriminant peut être nul ou négatif : toujours vérifier le signe de \(\Delta\) avant de calculer les racines.
  • La formule des racines nécessite \(\sqrt{\Delta}\) : vérifier que \(\Delta \geq 0\).

5. Forme factorisée

Propriété — Forme factorisée :
Si \(\Delta > 0\), la fonction \(f(x) = ax^2 + bx + c\) peut s'écrire sous forme factorisée : \[f(x) = a(x - x_1)(x - x_2)\] où \(x_1\) et \(x_2\) sont les racines.
Exemple — Vérification par développement

On a trouvé que \(f(x) = 2x^2 - 8x + 6\) a pour racines \(x_1 = 1\) et \(x_2 = 3\).

Forme factorisée : \(f(x) = 2(x-1)(x-3)\).

Vérification : \(2(x-1)(x-3) = 2(x^2 - 4x + 3) = 2x^2 - 8x + 6\). C'est bien \(f(x)\).

6. Tableau de signes

Propriété — Signe de \(f(x) = ax^2 + bx + c\) :
Le signe de \(f(x)\) dépend du signe de \(a\) et du discriminant \(\Delta\) :
  • Si \(\Delta > 0\) : \(f(x)\) est du signe de \(a\) à l'extérieur des racines et du signe opposé entre les racines.
  • Si \(\Delta = 0\) : \(f(x)\) est du signe de \(a\) pour tout \(x\) (et \(f(x) = 0\) uniquement en \(x_0\)).
  • Si \(\Delta < 0\) : \(f(x)\) est du signe de \(a\) pour tout \(x\) (jamais nul).
Méthode — Dresser le tableau de signes (cas \(\Delta > 0\)) :
1
Calculer les racines \(x_1\) et \(x_2\) (avec \(x_1 < x_2\)).
2
Placer \(x_1\) et \(x_2\) dans le tableau, avec \(0\) en-dessous.
3
À l'extérieur des racines : signe de \(a\). Entre les racines : signe opposé de \(a\).
Exemple — Tableau de signes de \(f(x) = 2x^2 - 8x + 6\)

\(a = 2 > 0\), racines : \(x_1 = 1\) et \(x_2 = 3\).

\(x\)\(-\infty\)\(1\)\(3\)\(+\infty\)
Signe de \(f(x)\)\(+\)\(0\)\(-\)\(0\)\(+\)

Donc \(f(x) \geq 0\) pour \(x \in\;]-\infty\;;\;1] \cup [3\;;\;+\infty[\).

7. Tableau de variations

Propriété — Variations d'une fonction du second degré :
  • Si \(a > 0\) : \(f\) est décroissante sur \(]-\infty\;;\;x_S]\) puis croissante sur \([x_S\;;\;+\infty[\). Le minimum est \(f(x_S)\).
  • Si \(a < 0\) : \(f\) est croissante sur \(]-\infty\;;\;x_S]\) puis décroissante sur \([x_S\;;\;+\infty[\). Le maximum est \(f(x_S)\).
Exemple — Variations de \(f(x) = 2x^2 - 8x + 6\)

\(a = 2 > 0\), sommet \(S(2\;;\;-2)\).

\(f\) est décroissante sur \(]-\infty\;;\;2]\) et croissante sur \([2\;;\;+\infty[\).

Le minimum de \(f\) est \(-2\), atteint en \(x = 2\).

8. Applications concrètes

Exemple 1 — Arche parabolique (charpentier)

L'arche de l'introduction est modélisée par \(h(x) = -x^2 + 4x\) sur \([0\;;\;4]\).

a) Pieds de l'arche : \(h(x) = 0 \iff -x^2 + 4x = 0 \iff x(-x + 4) = 0\), soit \(x = 0\) ou \(x = 4\). L'arche mesure 4 m de largeur au sol.

b) Hauteur maximale : \(x_S = \dfrac{-4}{2 \times (-1)} = 2\). \(h(2) = -4 + 8 = 4\) m. L'arche culmine à 4 mètres.

c) Passage d'un camion de 2,5 m de haut : On vérifie que \(h(x) \geq 2{,}5\) aux bords du camion (centré en \(x = 2\), largeur 2 m, donc bords en \(x = 1\) et \(x = 3\)).
\(h(1) = -1 + 4 = 3 \geq 2{,}5\) et \(h(3) = -9 + 12 = 3 \geq 2{,}5\). Le camion passe.

Exemple 2 — Seuil de rentabilité (menuisier agenceur)

Le bénéfice d'un artisan pour la fabrication de \(x\) meubles est \(B(x) = -2x^2 + 40x - 150\).

a) Seuils de rentabilité : \(B(x) = 0 \iff -2x^2 + 40x - 150 = 0 \iff x^2 - 20x + 75 = 0\).

\(\Delta = 400 - 300 = 100\), \(\sqrt{\Delta} = 10\).

\(x_1 = \dfrac{20 - 10}{2} = 5\) et \(x_2 = \dfrac{20 + 10}{2} = 15\).

L'artisan est rentable entre 5 et 15 meubles fabriqués.

b) Bénéfice maximal : \(x_S = \dfrac{-40}{2 \times (-2)} = 10\). \(B(10) = -200 + 400 - 150 = 50\) €.

Le bénéfice maximal est de 50 €, atteint pour 10 meubles.
