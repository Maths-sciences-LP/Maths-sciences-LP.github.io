Activité 3 – Comparer 2 tarifs : inéquation graphique SITUATION PRO

Chapitre 4 — Résolution graphique | 1ère Bac Pro | Mathématiques | ⏱ 30 min

Dernière mise à jour : 5 mai 2026, format manuel scolaire

Objectifs :
  • Résoudre graphiquement une inéquation
  • Trouver à partir de quand un tarif devient plus avantageux
  • Choisir entre forfait et tarif à l'unité

Situation – livraison de mobilier sur mesure

Marc, gérant de Mob'Bois à Lille, propose 2 tarifs de livraison : Tarif A = 50 € + 1,20 €/km. Tarif B = forfait 200 € quel que soit le km.

Problématique : À partir de quelle distance le forfait B est-il plus avantageux que A ?

Question 1 REA

Donner les expressions de f(x) = tarif A et g(x) = tarif B (x = distance en km).

f(x) = 50 + 1,20 x

g(x) = 200 (constant)

Question 2 REA

Calculer f(50), f(100), f(150), f(200) et tracer les deux droites.

f(50) = 110, f(100) = 170, f(150) = 230, f(200) = 290.

g(x) = 200 (droite horizontale).

Question 3 REA

Résoudre graphiquement et algébriquement f(x) = g(x).

50 + 1,20 x = 200 → 1,20 x = 150 → x = 125 km.

À 125 km, les deux tarifs sont égaux à 200 €.

Question 4 ANA

Pour x < 125 km, quel tarif est plus avantageux ?

Pour x > 125 km, quel tarif est plus avantageux ?

x < 125 km : f(x) < 200, le tarif A est moins cher.

x > 125 km : f(x) > 200, le forfait B est moins cher.

Question 5 REA

Pour une livraison à 80 km, lequel choisir ? À 200 km ?

80 km : f(80) = 50 + 96 = 146 €. Tarif A (146 € < 200 €).

200 km : f(200) = 290 €. Forfait B (200 € < 290 €).

Question 6 ANA

Marc propose un troisième tarif : C = 80 € + 1 €/km. Pour quel intervalle est-ce le moins cher des 3 ?

C(x) = 80 + x.

C vs A : 80 + x < 50 + 1,20 x → 30 < 0,20 x → x > 150. Donc C < A pour x > 150.

C vs B : 80 + x < 200 → x < 120. Donc C < B pour x < 120.

C n'est jamais simultanément le moins cher (intersection vide). C est toujours dominé par A ou par B.

Question 7 VAL

Marc fait 60 livraisons/an : 30 à moins de 50 km, 20 entre 50 et 150 km, 10 à plus de 150 km. Calculer le coût total avec une stratégie « tarif optimal pour chaque livraison ».

30 livraisons à < 50 km : tarif A. Coût moyen f(25) = 80 € → 30 × 80 = 2 400 €.

20 livraisons à 50-150 km : A jusqu'à 125 km, B au-delà. Estimation moyenne : 150 € → 20 × 150 = 3 000 €.

10 livraisons à > 150 km : B (200 €) → 10 × 200 = 2 000 €.

Total : 7 400 €.

Question 8 COM

Marc rédige sa grille tarifaire publique. 5 lignes max.

Tarifs livraison Mob'Bois
• Court trajet (≤ 125 km) : 50 € forfait + 1,20 €/km au-delà du domicile.
• Long trajet (> 125 km) : forfait fixe 200 € (toute distance).
• Bascule automatique au système le plus avantageux pour le client.
• Au-delà de 300 km : devis sur mesure.
• Livraison gratuite pour commande ≥ 2 500 €.

Pour aller plus loin (bonus)

Pour quelle distance les 3 tarifs (A, B, C) sont-ils égaux ?

A = B à x = 125 km, B = C à x = 120, A = C à x = 150. Pas de point commun aux trois. C est toujours dominé.

À retenir
  • Pour comparer 2 tarifs : tracer les fonctions et chercher les intersections.
  • L'inéquation f(x) < g(x) se résout graphiquement (zones où une courbe est sous l'autre).
  • Tarif fixe (forfait) : avantageux à grande distance/quantité. Tarif variable : avantageux à petite quantité.
